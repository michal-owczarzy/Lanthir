  1. Zkopírovat podpis — klikněte na modré tlačítko výše.
  2. Gmail: ⚙️ Nastavení → Zobrazit všechna nastavení → záložka Obecné → Podpis → Vytvořit nový → vložte (Ctrl+V) → Uložit změny.
  3. Outlook (web): Nastavení → Pošta → Vytvořit a odpovědět → vložte do pole podpisu → Uložit.
  4. Nastavte podpis jako výchozí pro nové zprávy i odpovědi.
